# FPGA Based Traffic Light Controller

This project implements a Traffic Light Controller using Verilog HDL.
It includes an emergency vehicle priority system.

## Features
- FPGA based design
- Emergency vehicle priority
- Written in Verilog HDL
- Simulated using Vivado / ModelSim

## Inputs
- Clock
- Reset
- Emergency Signal

## Outputs
- Red, Yellow, Green lights for two roads

## Tools Used
- Verilog HDL
- Xilinx Vivado
- ModelSim
